# Локальный поиск для задачи о независимом множестве
Проект реализует алгоритм локального поиска для нахождения приближенного решения задачи о максимальном независимом множестве в графе.

# Описание задачи
Независимое множество в графе — такое подмножество вершин, что никакие две вершины в этом подмножестве не соединены ребром. Задача о максимальном независимом множестве состоит в нахождении независимого множества наибольшего размера.

Эта задача является NP-трудной, поэтому для больших графов используются приближенные алгоритмы, такие как локальный поиск.

# Алгоритм
Локальный поиск для независимого множества
Инициализация: начинаем со случайного независимого множества

Исправление: удаляем вершины, нарушающие условие независимости

Улучшение: на каждой итерации пытаемся добавить вершины, не нарушающие независимость

Остановка: когда нельзя добавить ни одну вершину

# Как использовать:
Запустите программу

Выберите способ ввода:

1 - Ввести граф вручную

2 - Использовать готовый пример

# Пример ручного ввода (использован для проверки)
=== Поиск независимого множества ===
Выберите способ ввода графа:
1 - Ввести количество вершин и рёбра вручную
2 - Использовать готовый пример
Ваш выбор: 1
Введите количество вершин: 5
Введите количество рёбер: 4
Введите рёбра в формате 'u v' (вершины нумеруются с 0):

0 1

1 2

2 3

3 4

# РЕЗУЛЬТАТЫ
Найденное независимое множество: [0, 2, 4]
Размер: 3"""

O(maxIter × n²)

где:

n - количество вершин в графе

maxIter - максимальное количество итераций
